
# CulinaryCanvas - Personal Recipe API

## Description

This is a secure API for storing and managing personal recipes. It allows users to create, update, view, and delete their own recipes with user authentication and authorization.

## How to Run

1. Clone this repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file with the following:

```
MONGO_URI=<Your MongoDB URI>
JWT_SECRET=<Your JWT Secret>
```

4. Start the server: `npm run dev`.
5. The server will be running on `http://localhost:5000`.
